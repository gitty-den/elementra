# Elementra

Kreaturen-Sammel-Autobattler — sammle Elementarkreaturen, level sie mit Gold,
fusioniere sie zu Hybriden und besiege die 10 Stages der Kampagne.

**Spielen:** `index.html` doppelklicken (läuft ohne Server, ohne Installation).

- 🔥 Feuer schlägt 🌿 Natur schlägt 💧 Wasser schlägt 🔥 Feuer
- Kreatur antippen, wenn sie golden pulsiert → Spezialfähigkeit
- Zwei Kreaturen gleichen Archetyps auf Max-Level → Fusion zum neutralen Hybrid

Technik: reines HTML/CSS/JS, keine Dependencies, prozedurale SVG-Grafik,
WebAudio-Sound, Speicherstand im Browser (localStorage).

Projekt-Doku: `CLAUDE.md` (Architektur), `MASTERPLAN.md` (Roadmap),
`data/DATA_SCHEMA.md` (Datenmodell).
